# World File Cleaner – 自动清理存档文件

**World File Cleaner** 是一个简单的 Fabric 模组，用于在服务器关闭或单人存档退出时自动删除世界存档中不必要的文件，帮助保持存档文件夹整洁并减少磁盘占用。

## ✨ 功能特性
- 在专用服务器关闭或单人存档卸载时自动执行清理。
- 可配置删除的文件 / 文件夹（例如日志、缓存、会话锁、统计数据等）。
- 默认安全：仅删除非关键文件（除非您自行配置，否则不会删除区域文件或玩家数据）。
- 同时支持服务端和客户端（单人模式）。

## 📦 兼容性
- Minecraft：`1.21.11`
- 模组加载器：`Fabric`
- 使用 `ModMenu`

## ⚙️ 配置文件
首次运行后，会在以下路径生成配置文件：
```text
config/autoclean.json
```

配置示例：
```json
{
	"enabled": false,
	"deletePatterns": [],
	"whiteList": []
}
```

- `enabled`：设为 `false` 可完全禁用模组。
- `deletePatterns`：世界存档文件夹内要删除的文件，可用'*'作为通配符。
- `whiteList`：删除白名单，优先级高。

> 💡 **警告**：删除区域文件（*.mca）或 playerdata 会丢失游戏进度。默认模式已排除这些文件。

## ⚠️ 重要提示
- 首次使用自动删除功能前，请**务必备份**您的世界存档。
- 模组**不会**删除存档文件夹本身 – 只会删除其中的指定文件。
- 在专用服务器上，模组仅运行于服务端；客户端不需要安装，服务端清理仍可正常工作。
